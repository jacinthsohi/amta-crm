# Security Policy

Found a security issue in this repo? Please email me directly rather than opening a public issue:

**[jacinthsohi@gmail.com](mailto:jacinthsohi@gmail.com)**

I'll respond within a few days. If the issue affects the live deployment at [crm.mocktrial.tech](https://crm.mocktrial.tech), I'll work to patch it as quickly as possible.

## Scope

This is a hobby project running for a nonprofit. There's no bug bounty, but I genuinely appreciate responsible disclosure. Please include:

- The nature of the issue
- Steps to reproduce (if applicable)
- Whether it affects the live site or only the source code

Thanks for helping keep this project secure.
